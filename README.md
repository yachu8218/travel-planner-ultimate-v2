# Travel Planner Ultimate v3.4.0

## Live Share

建立新分享連結後，系統會保存一組只有分享者裝置知道的更新權杖。

- 修改行程、航班、預算或分帳後，約 1.4 秒自動同步。
- 分享中心可手動按「立即同步」。
- 親友端每 30 秒檢查一次。
- 有新版時才顯示「更新內容」。
- 接收者仍為唯讀，不能修改原始旅行。

> v3.3.0 以前建立的舊連結沒有更新權杖，需在 v3.4.0 重新建立一次 Live Share 連結，之後同一個連結即可持續同步。

## PDF 匯出中心

可自行選擇：

- 封面
- 旅行資訊
- 航班
- 每日行程
- 便條與待辦
- 預算摘要
- 錢包與分帳

預算與分帳預設不匯出。

樣式：

- App 手帳樣式
- A4 列印樣式

## Cloudflare

沿用原本的 `TRIP_SHARES` KV Binding，不需要新增其他設定。
