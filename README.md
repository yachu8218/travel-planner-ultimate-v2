# Travel Planner Ultimate v2.3.3

## 本次修正

- 全部頁面禁止橫向捲動
- 移除 v2.3.2 造成斷點與色差的 Header 延伸方式
- Header 使用單一主題色並固定與手機畫面同寬
- 錢包與翻譯所有內容自動縮排至手機寬度
- 點餐數量表取消固定最小寬度
- 快捷金額、翻譯分類、Day 分頁與天氣卡自動換行
- 整體卡片、按鈕、輸入框與間距縮小，單一畫面可看到更多資訊
- 保留目前所有旅行資料與功能

## 更新方式

解壓縮後將資料夾內全部內容上傳至原 GitHub Repository，覆蓋同名檔案並 Commit。Netlify 會自動部署。
