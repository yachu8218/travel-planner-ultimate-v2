# Travel Planner Ultimate v2.3.1

## 本次修正

### 匯率
改用三個公開匯率來源依序嘗試：

1. ExchangeRate-API 公開端點
2. Frankfurter
3. Currency API CDN
4. 全部失敗時使用最後一次成功快取

並加入韓元、日圓及其他幣別的常用金額快捷鍵。

### 旅行翻譯 Pro
- 依每趟旅行目的地，只顯示該目的地語言
- 韓國只顯示韓文、日本只顯示日文、泰國只顯示泰文
- 常用、餐廳、飯店、交通、購物、醫療、緊急各 10 句
- 點餐數量表支援 1～10
- 量詞：位、杯、份、個、瓶、碗、盤、串
- 點擊句子或數量可放大給店員看
- 慢速播放
- 一鍵複製
- 收藏常用句

## 更新方式

解壓縮 ZIP，將裡面全部內容拖入原本 GitHub Repository，覆蓋同名檔案後 Commit。Netlify 會自動部署。

本版不需要新增 Netlify API Key。
