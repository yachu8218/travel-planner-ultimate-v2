# Travel Planner Ultimate v3.3.0

## 完整旅行唯讀分享

分享時可選擇：

1. 僅行程
2. 行程＋預算
3. 行程＋完整錢包分帳

完整錢包分帳包含旅伴、消費明細、付款人、參與分帳者與最終結算。

親友開啟後仍為唯讀，不能新增、刪除或修改資料。

## 分享密碼

分享密碼為選填。設定後，親友開啟短網址時必須輸入正確密碼。

密碼只會以 SHA-256 雜湊儲存在 Cloudflare KV，不儲存明碼。

## Cloudflare

延用原本的 KV Binding：

```text
TRIP_SHARES
```

不需要新增其他環境變數或 Binding。
