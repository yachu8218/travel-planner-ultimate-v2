# Travel Planner Ultimate v2.3.0

## 旅行錢包
- 使用 Frankfurter 公開匯率服務，每日匯率，不需要 API Key。
- 支援雙向換算、手動重新整理與離線快取。
- 每趟旅行獨立設定旅伴、預算、消費與海外刷卡手續費。
- 每筆消費可指定付款人與參與人，並自動計算應收／應付。

## 旅行翻譯
- 使用 MyMemory 公開翻譯服務，不需要 API Key。
- 內建韓文與日文敬語常用句。
- 支援慢速語音播放、收藏句子與點餐數量速查。
- 機器翻譯不保證完全符合敬語，重要情境建議使用內建敬語句。

## 更新方式
解壓縮後，全選內容並上傳至原本 GitHub Repository，覆蓋同名檔案後 Commit。Netlify 會自動部署。
