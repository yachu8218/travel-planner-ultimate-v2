# Travel Planner Ultimate v2.7.0 — 景點收藏中心

## 新增功能

- 行程卡右上角「⋯」中的收藏功能正式啟用
- 每趟旅行擁有獨立收藏清單
- 探索頁新增「搜尋地點」與「我的收藏」分頁
- Google Places 搜尋結果可收藏
- 收藏項目可一鍵加入目前 Day
- 可開啟地圖導航或刪除收藏
- 收藏保留店名、照片、地址、評分、營業狀態等資料

## 更新方式

解壓縮後覆蓋 GitHub 原檔案並 Commit，Cloudflare Pages 會自動部署。

需保留 Cloudflare Secret：

```text
GOOGLE_PLACES_API_KEY
```
