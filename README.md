# Travel Planner Ultimate v2.6.2 — Smart Place Card UI

## 本次更新

地點、餐廳、甜點與住宿行程卡改為智慧地點卡：

- 店名使用較大、較粗且不同顏色的文字
- 類別標籤
- 評分、評論數與營業狀態標籤
- 今日營業時間摘要
- 完整營業時間可展開
- 地址可展開完整內容
- Google Maps／NAVER Map 快捷導航
- 電話可直接撥號
- Google Places 店家照片

## 更新方式

解壓縮後覆蓋 GitHub 原檔案並 Commit，Cloudflare Pages 會自動部署。

需保留既有 Secret：

```text
GOOGLE_PLACES_API_KEY
```
