# Travel Planner Ultimate v2.8.3 — 航班提醒與報到中心

## 新增功能
- 航班起飛倒數
- 建議抵達機場時間
- 線上報到時間提示
- 登機前準備進度
- 一鍵下載 `.ics` 加入 iPhone／Google 行事曆
- 行事曆事件包含航班、航廈、Gate、行李轉盤與機型
- 事件預設在起飛前三小時提醒
- 首頁顯示最近航班與出發倒數
