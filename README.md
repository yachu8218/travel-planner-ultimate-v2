# Travel Planner Ultimate v3.0.1

## 本次修正

### 首爾地鐵搜尋
擴充首爾 1～9 號線與機場鐵路站名，可使用中文、韓文或英文搜尋。

### 地鐵時間
不再將所有車站固定計算為每站 3 分鐘，改依路線類型、站數、起始候車時間及轉乘步行時間估算。

> App 內仍屬旅行規劃估算。實際班距、臨時營運狀況與即時抵達時間，請以 Google Maps、Naver Map 或當地營運資訊為準。

### 天氣中心
「我的」頁面的天氣中心與首頁相同，固定顯示完整 7 天。舊版留下的 4 天快取會自動重新取得。

### 航班行程卡
航廈、登機門、行李轉盤與機型只顯示於白底資訊框，上方機場路線區不再重複。

## 更新方式
解壓縮後覆蓋 GitHub 原檔案並 Commit，Cloudflare Pages 會自動部署。原有 Secrets 不需修改。
